# Frontend - AI Quiz & Chat App

This is the React + Vite frontend for an AI-powered learning platform.

## Features

- AI Chat Assistant
- AI Quiz Generation
- Quiz History Tracking
- Multi-language Support
- JWT Authentication

---

## Tech Stack

- React (Vite)
- JavaScript
- Fetch API
- CSS (custom styling)
