# Java Basics

## What is Java?

Java is an object-oriented, class-based programming language designed for portability and scalability.

Java applications run on the Java Virtual Machine (JVM).

---

## JVM, JRE, and JDK

- JVM executes Java bytecode
- JRE provides runtime environment
- JDK includes development tools

---

## Classes and Objects

Java uses classes to define objects.

Example:

```java
class Car {
    String brand;
}
```

Objects are instances of classes.

---

## OOP Concepts

Java supports:
- encapsulation
- inheritance
- polymorphism
- abstraction

These concepts improve maintainability and scalability.

---

## Exception Handling

Java uses try-catch blocks for exception handling.

Example:

```java
try {
    int result = 10 / 0;
} catch (Exception e) {
    System.out.println("Error");
}
```

---

## Collections Framework

Java provides:
- ArrayList
- HashMap
- HashSet
- Queue

These data structures simplify data management.

---

## Multithreading

Java supports multithreading for concurrent execution.

Threads improve performance for CPU-intensive applications.

---

## Unique Fact

Java follows the principle:
"Write Once, Run Anywhere."
